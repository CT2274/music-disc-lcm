Minecraft Music Disc LCM Calculator
How to use:
1. Select music disc from dropdown
2. Input the off duration of the jukebox (how long it takes for the hoppers/droppers to reset it)
3. Set the offset. This is how long the jukebox takes to start playing once the player gives the first input.
Results:
LCM: The Least Common Multiple between all periods. Longest total cycle where every clock has returned to its starting offset (not necessarily silent) First Silence: How many ticks until all jukeboxes stop playing simultaneously. Smallest tick value where every clock is in its “off” window at the same time.
